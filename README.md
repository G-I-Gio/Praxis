## 🇫🇷 Français

Praxis est un fork de [Razzia](https://github.com/Ralex91/Razzia), plateforme de quiz open-source et auto-hébergée. Il étend le projet original avec un système d'authentification multi-utilisateurs, une API REST complète, un tableau de bord avancé et une configuration pensée pour un déploiement en entreprise.

**[📖 Lire la documentation française →](docs/README.md)**

## 🇬🇧 English

Praxis is a fork of [Razzia](https://github.com/Ralex91/Razzia), an open-source self-hosted quiz platform. It extends the original project with a multi-user authentication system, a full REST API, an advanced dashboard and configuration designed for enterprise deployment.
