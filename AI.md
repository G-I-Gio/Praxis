## 🇫🇷 Développement assisté par IA

Ce projet a été développé en collaboration avec **Claude** (Anthropic) via [claude.ai](https://claude.ai).

### Contexte

Praxis est né de la curiosité d'un bidouilleur passionné d'électronique et d'informatique, habitué à mettre les mains dans les systèmes — des microcontrôleurs aux serveurs, en passant par les réseaux et l'auto-hébergement. Pas développeur de formation, mais à l'aise pour lire, comprendre et raisonner sur du code dans de nombreux langages.

L'architecture, les choix techniques, les décisions de conception et les validations fonctionnelles ont été pilotés humainement. Claude a servi d'outil d'exécution — traduire les intentions en code, déboguer, structurer la documentation.

### Ce que ça signifie concrètement

- Les choix d'architecture sont intentionnels et réfléchis
- Le code est fonctionnel et testé manuellement sur plusieurs environnements
- La documentation reflète fidèlement ce qui est implémenté
- Ce projet est proposé en toute transparence, sans prétendre à une expertise de développeur professionnel

### Outil utilisé

**Claude** par Anthropic · [claude.ai](https://claude.ai)

## 🇬🇧 AI-Assisted Development

This project was developed in collaboration with **Claude** (Anthropic) via [claude.ai](https://claude.ai).

### Context

Praxis was born from the curiosity of an electronics and computing enthusiast — someone who enjoys tinkering with systems at every level, from microcontrollers to servers, networks and self-hosting. Not a trained developer, but comfortable reading, understanding and reasoning about code across many languages.

The architecture, technical choices, design decisions and functional validation were all driven by the human side. Claude served as an execution tool — translating intentions into code, debugging, and structuring documentation.

### What this means in practice

- Architectural choices are intentional and thought through
- The code is functional and manually tested across multiple environments
- The documentation accurately reflects what is implemented
- This project is offered in full transparency, without claiming professional developer expertise

### Tool used

**Claude** by Anthropic · [claude.ai](https://claude.ai)
